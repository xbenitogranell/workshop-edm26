# workshop-edm26
Repository containing scripts and data for the workshop "diatom ecology through lenses of OPEN and FAIR practices" in Murcia, Spain (6-7-2026) -- 16th European Diatom Meeting

# Database
The SeeD diatom database is the result of a networking project funded by the International Society for Diatom Research (ISDR). The objectives are:
a. to improve the quality of open-access diatom data for enabling large-scale ecological synthesis, with special emphasis in Iberoamerican inland waters
b. build trust within a community of data generators and stewards
c. inspire the next generation of diatom ecologists to create, share, and utilize global spatio-temporal data, 
d. stimulate networking within the Iberoamerican diatom researchers.
